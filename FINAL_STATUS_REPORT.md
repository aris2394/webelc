## 📞 Support & Maintenance

### Making Changes
1. **Edit Page Content**: Modify `.tsx` files in `/src/app/`
2. **Update Styles**: Use Tailwind classes in components
3. **Add Images**: Place in `/public/` folder
4. **Update Navigation**: Edit `Navbar.tsx` and `Footer.tsx`

### Rebuilding
```bash
npm run build    # Production build
npm run dev      # Development server
npm run lint     # Code quality check
npm start        # Production server
```
